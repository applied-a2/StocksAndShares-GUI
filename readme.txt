Project: "Stocks and shares"
Authors: Peter Hearne, Shane Halley, Ian Barnes, Abdullahi Shafii, Thai Kha Le

You need to have Java 8 and e(fx)eclipse to run this project. 2 main features are:
-Playable 2 versions (console or javafx) via only 1 method call in 1 class.
-Game modification is available, data is then stored in xml file for afterward plays.
Main method to run is in class MainApp.java, package sns.driver
